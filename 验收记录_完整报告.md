## 发现的问题汇总

### 任务一
1. **pause/resume/stop 返回 400**: 所有模型在任务快速失败后都返回 400 — 这是正确行为，不是 bug
2. **Sigma /types 格式不一致**: 返回 list 而非 dict，与前端期望的格式不匹配
3. **Sigma 缺暗色模式**: 前端无 `dark:` CSS 类
4. **Quartz 的 /types 不含 tool_chaining**: 用了中文分类名映射，与 Raven/Prism 的英文名不一致

### 任务二
1. **所有模型后端还原为原始版本**: attack_loop.py 从 1087→468 行，无 /types 端点，前端会报 404
2. **需要混用任务一后端**: 测试任务二前端时必须使用任务一 Raven 的后端（有完整端点）
